# AI-Number-Guessing-Challenge-Game

## 🌌 Overview
This project is a futuristic AI-based Number Guessing Game built using HTML, CSS, and JavaScript.  
The game challenges the user to guess a randomly generated number between 1 and 100 with the help of AI-style feedback.

It combines logic, UI design, and interactive animations to create an engaging experience.

---

## 🚀 Features
- Random number generation (1–100)  
- AI-based feedback system (Too High / Too Low)  
- Attempt counter  
- Rank system based on performance  
- Neon futuristic UI with glassmorphism effect  
- Animated AI avatar with dynamic expressions  
- Celebration effects (particles/confetti) on winning  
- Restart game option  
- Enter key support for quick input  
- Fully responsive design  

---

## 🕹️ How to Play
1. Enter a number between 1 and 100  
2. Click "EXECUTE GUESS" or press Enter  
3. Follow hints:
   - Too High → Try a smaller number  
   - Too Low → Try a larger number  
4. Keep guessing until you find the correct number  
5. Win the game and get your rank  

---
